# Imágenes de los Socios Fundadores

Por favor, descarga estas imágenes manualmente y colócalas en esta carpeta:

## Imágenes necesarias:

1. **daniel-garcia.jpg**
   - URL: https://horizoninmobiliaria.com/wp-content/uploads/2022/05/daniel-garcia-horizon-inmobiliaria.jpg
   - Descarga: Haz clic derecho en el enlace → "Guardar imagen como..." → Guarda como `daniel-garcia.jpg`

2. **christian-vela.jpg**
   - URL: https://horizoninmobiliaria.com/wp-content/uploads/2022/05/christian-vela-horizon-inmobiliaria.jpg
   - Descarga: Haz clic derecho en el enlace → "Guardar imagen como..." → Guarda como `christian-vela.jpg`

## Ubicación final:

Las imágenes deben estar en: `public/images/founders/`

```
public/
└── images/
    └── founders/
        ├── daniel-garcia.jpg
        └── christian-vela.jpg
```

Una vez descargadas, las imágenes aparecerán automáticamente en la sección "Nosotros" del sitio web.
